On effectue des changements sur le document précédemment envoyé (vide) sur Github

Comment ajouter ces changements sur Github

  • On sauvegarde ses changements pour commencer
  • Dans le terminal de commandes, on se place dans le bon dossier (ici 03_git) et on va taper la commande "git add exercice.html" (j'ai modifié le nom afin de rendre ça plus clair)
  • Une fois qu'on a ajouté notre fichier, on peut faire notre commit afin de notifier des changements. On utilise la commande "git commit -m"modification de l'exercice 03_git""
  • Ensuite, pour envoyer le tout sur Github on va utiliser la commande "git push"
